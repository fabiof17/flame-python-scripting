# reconstructCycles

Reconstruct Blender beauty pass from exr.

Used passes :
- DiffCol
- DiffDir
- DiffInd

- GlossCol
- GlossDir
- GlossInd

- SubsurfaceCol
- SubsurfaceInd
- SubsurfaceInd

- TransCol
- TransDir
- TransInd

- Emit
- Env

### HOW TO USE IT

* Select a Clip node.
* Context menu.
* PYTHON: NODES: Reconstruct Blender Beauty

### INSTALLATION

* Copy reconstructCycles.py to '/opt/Autodesk/shared/python' folder.
